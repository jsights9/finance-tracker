# 💰 Finance Tracker App

A simple and responsive finance tracking application built with React. This app allows users to manage their expenses, track spending by category, and persist data using local storage.

---

## 🚀 Features

* Add new expenses with name, amount, and category
* Edit existing expenses
* Delete individual expenses
* Clear all expenses
* View total spending
* View spending by category
* Form validation with user-friendly error messages
* Data persistence using localStorage
* Clean and responsive UI

---

## 🛠️ Tech Stack

* React (useState)
* JavaScript (ES6+)
* CSS
* LocalStorage API

1. Clone the repository
2. Navigate to the project folder
3. Install dependencies:
   npm install
4. Start the development server:
   npm run dev
